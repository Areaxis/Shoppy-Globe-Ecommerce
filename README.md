# ShoppyGlobe E-Commerce (React & Vite)

A simple and interactive e-commerce web application built using React and Redux Toolkit.
This project allows users to browse products, search items, view product details, manage a shopping cart, and place a dummy order with a clean and responsive user interface.

## Libraries Used

* React (Vite)
* Redux Toolkit
* React Router DOM

## Features

* Browse products from an external API
* Search products by title
* View detailed information of each product
* Add products to cart
* Increase or decrease product quantity in cart
* Remove products from cart
* Checkout form with user details
* Order summary showing selected products
* Order placement with automatic redirect to home
* Dynamic routing using React Router
* Global state management using Redux Toolkit
* Lazy loading for components and images
* Responsive design for mobile and desktop
* 404 page for invalid routes with error details

## How to install

Use "npm install" in order to installed the required dependencies and run "npm run dev" in order to begin hosting in your localhost.
